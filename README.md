# CustomStackSize

Customize the stack size of different resources in Factorio.

- Brick
- Concrete
- Hazard concrete
- Refined concrete
- Refined hazard concrete
- Landfill
- Wood
- Ice platforms
- Foundations
- Artificial jellynut soil
- Artificial yumako soil
- Iron ore
- Copper ore
- Stone
- Coal
- Uranium ore
- Raw fish
- Ice

## Important Notes

- Each type can have its own stack size. Go to Mod settings -> Startup
- Works with already running saves
- Make backups of your saves just to be sure. I don't want to be involved in you loosing a billion hours on your save
